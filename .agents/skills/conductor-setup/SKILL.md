# conductor-setup

Initialize the Conductor SDD project structure.

## When to Use
- First time setting up SDD workflow in a project
- When conductor/ directory structure is missing
- When project needs spec-driven development setup

## What It Does
- Creates conductor/ directory structure
- Sets up templates for tracks, specs, and plans
- Configures project for SDD workflow

## Usage
Run this skill before using other conductor skills if the project structure doesn't exist.
