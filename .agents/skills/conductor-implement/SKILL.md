# conductor-implement

Implement a selected track after the spec and tasks are clear.

## When to Use
- Track has been created and approved
- Ready to start coding
- Tasks are clearly defined

## What It Does
- Implements the track one task at a time
- Validates each change
- Stays within spec boundaries

## Usage
Run this skill to implement an approved track. Do not make changes outside the spec without notifying the user.
