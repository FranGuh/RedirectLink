# conductor-newTrack

Create a new feature/bugfix/refactor track.

## When to Use
- Starting a new feature implementation
- Planning a bug fix
- Beginning a refactoring effort
- Any ambiguous or complex change

## What It Does
- Creates a new track with spec, plan, tasks, risks, and acceptance criteria
- Documents requirements before coding
- Establishes clear boundaries for implementation

## Usage
Run this skill to create a track before implementation. Use after recovering Engram context.
